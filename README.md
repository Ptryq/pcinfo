# pcinfo

To install Bun:

Linux 🐧:
```bash
curl -fsSL https://bun.sh/install | bash
```
                 🪟
Windows 🏌️:
```bash
powershell -c "irm bun.sh/install.ps1|iex"
```

To install dependencies:

```bash
bun install
```

To test client run:

```bash
bun run index.ts
```

To run server:

```bash
bun run server.ts
```

To build executable client run (i hope this work 💀💀💀):

```bash
bun run build
```
